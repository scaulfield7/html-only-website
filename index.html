HTML-Only Website In Dark Mode

Hi there. You're probably wondering why I built this website using only HTML. Fortunately, due to the fast loading speed of HTML-only websites, you don't need to wait long to find out.

Other than wanting to set myself a fun design challenge, below are the main reasons.

Why HTML?

  • Loads quickly
  • Minimalist design (no distractions)
  • Sustainable
  • Compatible in older browsers
  • Compatible with multiple devices

Also, it's old school, and nostalgia is pretty in right now.
